📋 GoT 고급 프롬프트 복사

아래 버튼을 눌러 프롬프트를 복사하세요.
.

우리 시에서 운영 중인 스마트시티 통합플랫폼의 시민 이용률이 
지난 6개월간 계속 감소하고 있습니다. 
현재 월평균 이용자 수는 12,000명 수준이며, 시민 만족도 조사 결과 
'사용이 어렵다'는 응답이 45%입니다. 이용률 개선 방안을 제시해주세요.

이 문제를 해결하기 위해 다음과 같이 **상호 연결된 다각도 분석**을 수행해주세요:

═══════════════════════════════════════════════════════════

【노드 A: 기술적 요인 분석】
├─ A1: UI/UX 복잡도 문제
├─ A2: 시스템 성능 및 속도 이슈
├─ A3: 모바일/웹 호환성 문제
└─ A4: 접근성(장애인, 고령자) 부족

【노드 B: 콘텐츠/서비스 요인 분석】
├─ B1: 제공 서비스의 실용성 부족
├─ B2: 정보 최신성 및 정확도 문제
├─ B3: 타 플랫폼 대비 차별성 결여
└─ B4: 서비스 카테고리 구성 비직관성

【노드 C: 사용자 행태 요인 분석】
├─ C1: 연령대별 디지털 리터러시 격차
├─ C2: 플랫폼 인지도 및 홍보 부족
├─ C3: 사용 동기 부여 요소 결여
└─ C4: 기존 이용자 이탈 패턴

【노드 D: 조직/운영 요인 분석】
├─ D1: 부서 간 데이터 연계 미흡
├─ D2: 사용자 피드백 수렴 체계 부재
├─ D3: 유지보수 및 업데이트 주기 문제
└─ D4: 예산 및 인력 배분의 적절성

═══════════════════════════════════════════════════════════

**1단계: 노드 간 연결관계 파악**
각 노드(A, B, C, D)의 세부 요인들이 서로 어떻게 영향을 주고받는지
연결 관계를 분석해주세요.

예시:
- A1(UI/UX 복잡도) ←→ C1(디지털 리터러시): 어떤 상관관계?
- B1(서비스 실용성) ←→ C3(사용 동기): 어떻게 연결되는가?
- D1(데이터 연계) ←→ B2(정보 최신성): 어떤 인과관계?

**2단계: 핵심 병목 지점 식별**
위에서 파악한 연결 관계를 바탕으로, 가장 많은 다른 요인들에게 
부정적 영향을 미치는 "핵심 병목 지점" 3가지를 찾아주세요.
각 병목 지점이 연결된 다른 노드들을 명시해주세요.

**3단계: 다중 경로 솔루션 설계**
각 핵심 병목 지점에 대해, 해결 시 여러 노드에 동시 긍정적 영향을 
줄 수 있는 "파급효과 큰 솔루션"을 제안해주세요.

각 솔루션마다:
✓ 직접 해결되는 노드: [    ]
✓ 간접 개선되는 노드: [    ]
✓ 예상 파급 경로: [노드A] → [노드B] → [노드C]
✓ 시너지 효과 점수: (1-10점)

**4단계: 최적 실행 순서 도출**
노드 간 의존성을 고려하여, 어떤 솔루션을 먼저 실행해야
다른 개선 작업의 효과를 극대화할 수 있는지 
"실행 우선순위 그래프"를 그려주세요.

형식:
[1순위 솔루션] 
    ↓ (이것이 해결되면)
[2순위 솔루션] ← 이 솔루션의 효과 2배 증폭
    ↓
[3순위 솔루션] ← 이 솔루션의 리스크 50% 감소

**5단계: 순환 참조 및 트레이드오프 분석**
- 서로 상충되는 솔루션 조합이 있는가? (예: A 해결 시 B 악화)
- 예산/인력 제약 조건에서 선택해야 할 경로는?
- 각 경로 선택 시 포기해야 하는 개선 효과는?

**6단계: 통합 로드맵 제시**
위의 그래프 분석을 종합하여:
├─ Phase 1 (1-3개월): 핵심 병목 해소 → 연쇄 개선 효과 [  ]
├─ Phase 2 (4-6개월): 파급효과 극대화 → 추가 개선 효과 [  ]
└─ Phase 3 (7-12개월): 최적화 및 고도화 → 최종 목표 달성

각 Phase별로 개선이 전파되는 "노드 활성화 경로"를 
시각적으로 표현해주세요.
